## Le contexte

Une PME industrielle gérait ses plannings, ses stocks et ses interventions
terrain dans trois fichiers Excel partagés sur un drive. Synchronisation
manuelle, erreurs fréquentes, données jamais à jour.

L'objectif : un outil unique, accessible depuis le terrain, sans formation.

## Le défi

Le vrai défi n'était pas technique — c'était de comprendre un métier complexe
avec des contraintes spécifiques : techniciens sur le terrain avec des téléphones,
responsables en bureau, données sensibles. Chaque processus avait son exception.

## La solution

Une application Rails avec Hotwire — pas d'app mobile native, mais une PWA
responsive qui fonctionne sur les téléphones des techniciens sans installation.
Turbo Streams pour les mises à jour temps réel entre les utilisateurs.
Une interface pensée pour être utilisée avec des gants.

## Le résultat

Déployée en 8 semaines. Adoption immédiate — les équipes terrain ont arrêté
d'utiliser WhatsApp pour se coordonner. Zéro formation nécessaire.
Le client a demandé trois modules supplémentaires dans le mois suivant la livraison.
